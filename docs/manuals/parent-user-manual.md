# Parent User Manual

## Who this is for
Parents and caregivers who use Family Hero Hub to manage children, points, rewards, routines, school prep, savings-style goals, and linked child devices.

## What this manual covers
- Signing in with Google
- Requesting access if needed
- Understanding the parent dashboard
- Adding and managing children
- Awarding and removing points
- Creating and managing rewards
- Reviewing reward requests
- Using saved points and allowance-style values
- Using the calendar
- Using School Prep / School Bag
- Linking and unlinking child devices
- Managing caregivers and family access

## 1) What Family Hero Hub does
Family Hero Hub helps your family keep track of everyday responsibility in one private, parent-led space.

In simple terms:

- parents set up the family
- parents manage children, points, rewards, and family settings
- children can request rewards from their own dashboard
- parents approve or reject those requests
- families can optionally use allowance-style point values
- parents can manage calendar items and school prep

## 2) How parents sign in

1. Open the login page.
2. Choose **Continue with Google**.
3. Sign in with the Google account approved for your family.
4. After sign-in, you go to the parent dashboard.

### Google login in plain English

- Google handles the secure sign-in screen.
- You do **not** type your Google password into Family Hero Hub.
- Family Hero Hub only receives the sign-in result.
- Parent sessions stay active for convenience.
- Always log out on shared devices or any device you do not control.

## 3) Request access if you are new

If your account is not approved yet:

1. Go to **Request access**.
2. Enter your name, email, family name, and optional message.
3. Submit the request.
4. Wait for approval.
5. After approval, sign in with Google.

## 4) Parent dashboard overview

The parent dashboard is your family overview.

You will see:

- child avatar buttons with point badges and names
- a secondary **Settings** menu
- an **Add Child** button

### What the child buttons show

Each child button shows:

- an avatar image when available, or initials fallback
- a point badge showing the child's current available points
- the child name underneath

Tap a child button to open the child action modal.

### Child action modal

The child action modal opens on **Points** by default.

From there you can:

- add points
- remove points
- use a custom point action
- open the child dashboard
- edit the child name and avatar
- review reward requests
- view School Bag
- view Calendar
- review Savings
- check the Points Log

## 5) Adding and managing children

Use **Add Child** on the parent dashboard to create a child.

From the child action modal, parents can:

- add or remove points
- open the child dashboard
- edit child name and avatar
- review that child's requests
- view School Bag, Calendar, Savings, and Points Log

Use **Settings** for secondary family tools:

- manage rewards
- allowance setup
- family settings or member controls
- link a child device
- manage caregivers and invites

## 6) Adding and removing points

Open the child action modal for a child, then stay on **Points**.

You can choose:

- **Add points**: add points for helpful actions, routines, or effort
- **Remove points**: remove points when the family needs to reset expectations
- **Custom**: use a custom one-time action

### Add points

Use this when a child does something helpful, kind, or responsible.

### Remove points

Use this when you need to remove points to reflect family expectations or missed responsibilities.

### Custom point actions

Use **Custom** when the action does not fit the preset buttons.

Write a short reason so the record is clear later.

### Edit a child

Use **Edit Child** in the child action modal when you need to:

- change the display name
- choose a different avatar

## 7) Points balance, saved points, and points on hold

The app shows three point balances for each child:

- **Available points**: points the child can use right now
- **Saved points**: points set aside for later. Saved points are a family feature, not banking. They can unlock later with a bonus.
- **Points on hold**: points temporarily set aside while a reward request waits for your decision

The points log shows the history of point changes, savings deposits, savings unlocks, bonus awards, and reward requests.

### Savings and allowance

Family Hero Hub can optionally show allowance-style values beside points.

- saved points are still family points
- allowance values are optional and parent-managed
- Family Hero Hub does not hold money, transfer money, or pay children
- parents remain responsible for any real-world allowance, purchase, or reward

Use points to encourage effort and follow-through, not just payment for chores. Some families connect points to allowance values, while others use points only for rewards, privileges, or progress.

When paid subscriptions are available, Family Hero Hub plans to use Apple App Store and Google Play subscriptions for consumer billing. Payment method details are handled by the app store you use. Family Hero Hub does not collect or store payment card or bank account details.

## 8) Rewards overview

Rewards are things a child can earn with points.

A reward has:

- title
- description
- icon or emoji
- point cost
- active / inactive status

Parents create rewards, and children can see the active rewards that are available to them.

## 9) Creating and managing rewards

From **Settings**, open **Manage rewards**.

You can:

- create a reward
- edit an existing reward
- set the reward cost
- mark rewards visible or hidden
- keep rewards simple and clear for children

Good reward examples:

- extra screen time
- choose dessert
- special game time
- family movie night

## 10) Viewing reward requests

1. Your child requests a reward.
2. The points are put on hold from the available balance.
3. You approve or reject the request.
4. If you approve it, the reward is completed and the points stay used.
5. If you reject it, the points on hold return to the child's available points.

## 11) Calendar, School Prep, and School Bag

Use the calendar for family tasks and events.

School Prep is the parent view for today and tomorrow's school items.
School Bag is the child view that shows the school items to remember.

## 12) Linking and unlinking child devices

Use the child device link tools when you want a child dashboard on a tablet or phone.

- linking connects a device to a child dashboard
- unlinking signs that device out without deleting the child profile or history

## 13) Parents and caregivers

Family Hero Hub lets parents invite trusted grownups and remove access when needed.

Removing a caregiver removes access without deleting the family data.

## 14) Troubleshooting

- If sign-in fails, confirm you are using the approved Google account.
- If access is still pending, wait for approval or recheck the email you submitted.
- If a child device will not open, refresh the link or ask for a fresh one.
- If points or rewards look wrong, check the recent point history and reward requests.
- If a calendar or School Bag item is missing, check the child and the date.
