# Messaging v1 training video storyboards

Each script targets 60–120 seconds, uses demo data only, avoids message bodies in thumbnails/logs, and needs Fusha narration/caption review before Arabic release.

1. **Teacher sends and replies:** open Messages; confirm context; send neutral text; explain idempotent retry and emergency boundary; show Sent/Delivered/Read.
2. **Parent School Chats:** open child; select School Chats; reply; show read-only/closed guidance; explain school-controlled access.
3. **Photo and voice:** choose safe photo; explain metadata removal and protected access; record/listen/send voice; show unavailable-state support path.
4. **Receipts:** animate committed → delivered → read; state policy dependency; explain what each status does and does not prove.
5. **Contact hours:** show after-hours send, held generic notification, next-window release, and responsible urgent use.
6. **Safeguarding review:** open protected area; reason and acknowledgement; timer/audit banner; show isolation from participant receipts/unread.
7. **Restricting/closing:** choose least restrictive control; confidential versus neutral wording; confirm; show audit and reopening approval.
8. **Evidence export:** request; show queued/generating/verified; download; record hashes; run verification utility; explain expiry/download limit.
9. **Granting access:** select active staff; compare permissions; grant minimum with reason; revoke; explain roles do not imply access.
10. **System Owner handover:** show current owner definition; select another active admin twice; enter reason; warning; transfer; show history and former-owner limits.
